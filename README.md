# react-fileupload

simple fileupload by
* client with react and axios
* server with express and multer

## Installation
```
$ npm i

# if you  don't have parcel
$ npm i -g parcel-bundler

# build coffeescript
$ npm run coffee

# build index.js
$ npm run build

# run express by gulp
$ npm run devel
```
